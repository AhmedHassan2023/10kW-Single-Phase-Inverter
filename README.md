# 10kW Single-Phase Industrial Inverter

## Overview

This project presents the design and implementation of a 10 kW single-phase full-bridge inverter intended for industrial UPS and power conversion applications.
10kW Single-Phase Full-Bridge Inverter with SVPWM, PLL Grid Synchronization and Static Bypass using STM32F446.

## Specifications

| Parameter | Value |
|-----------|-------|
| Input Voltage | 400 VDC |
| Output Voltage | 220 VAC |
| Rated Power | 10 kW |
| Switching Frequency | 16 kHz |
| Controller | STM32F446VET6 |
| Modulation | SPWM / SVPWM |

## Key Features

- Single-phase full-bridge topology
- SVPWM implementation
- Grid synchronization using PLL
- Static bypass functionality
- Closed-loop voltage regulation
- Multi-layer PCB design
