# Hardware

This folder contains hardware-related resources for the 10 kW single-phase inverter project.

## Contents

- Control PCB layouts and 3D views
- Power PCB layouts and 3D views
- Assembled hardware photos


## Hardware Features

- 4-layer control PCB
- 4 oz copper power PCB
- Isolated gate drivers
- STM32F446VET6 controller
- High-current power stage
